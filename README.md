# MacCms综合内容网站模板_代号：t3

#### 介绍
基于MacCmsV10为内核的综合类UI模板，配置简单，不修改CMS内核，不覆盖任何原装文件！上传即可使用！

#### 软件架构
Html5+Css+js+Jq


#### 模板功能

全局PC与Web端响应式
基于bootstrap元素设计
内容活跃完善化自控
视频类内容展示与下载
文学类内容展示与下载
待开发增加
